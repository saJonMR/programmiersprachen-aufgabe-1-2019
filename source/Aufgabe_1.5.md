#  Aufgabe 1.5

### Initialisierung:
Eine Variable bekommt den ersten Wert zugeteilt.

#### Bsp:
```
int zahl;
```

### Zuweisung:
Eine Variable bekommt einen Wert zugewiesen.

#### Bsp:
```
int zahl;
zahl = 11;
```

### Unterschied:
Die Initialisierung beschreibt also die erstmalige Zuweisung eines Wertes zu einer Variable.
